# Сайт-портфолио фотографа

Лёгкий статический сайт-портфолио для фотографа: тёмная/светлая тема, приветственный экран, фильтрация портфолио, просмотр фото в модальном окне и форма контакта без сервера (через `mailto:`).

## Быстрый старт

- Откройте `index.html` двойным кликом (или через Live Server в VS Code/Cursor).

## Технологии

- HTML5
- CSS3 (CSS-переменные, адаптив, плавные переходы тем)
- Vanilla JavaScript (без фреймворков)

## Структура

- `index.html` — разметка сайта
- `styles.css` — стили (включая темы и анимации)
- `script.js` — интерактив: модалка, форма, темы, welcome-экран и т.п.

## Возможности

- Переключение темы (тёмная/светлая) с сохранением выбора
- Плавная анимация при смене темы
- Просмотр фото в модальном окне
- Навигация по фото стрелками `←` / `→` в открытой модалке
- Приветственный экран (можно не показывать снова)
- Фильтры портфолио по категориям
- Контактная форма без бэкенда (открывает почтовый клиент)

## Клавиши

- `Esc` — закрыть приветственный экран/модалку
- `←` / `→` — переключение фото в модальном окне

## Репозиторные файлы
