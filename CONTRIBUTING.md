# Contributing

Спасибо за интерес к проекту.

## Локальная работа

1. Откройте проект в Cursor/VS Code.
2. Запустите через Live Server или просто откройте `index.html`.
3. Вносите изменения небольшими, логически завершёнными частями.

## Стиль изменений

- Сохраняйте текущую структуру файлов: `index.html`, `styles.css`, `script.js`.
- Пишите понятные классы и комментарии к сложным HTML-блокам.
- Для новых UI-элементов учитывайте доступность: `aria-label`, управление с клавиатуры.

## Проверка перед коммитом

- Проверьте сайт в тёмной и светлой теме.
- Проверьте приветственный экран и модальное окно (включая клавиши `Esc`, `←`, `→`).
- Убедитесь, что фильтры портфолио корректно скрывают/показывают карточки.
